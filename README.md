# Pump Sound Classification

This project explores anomaly detection for industrial pump sounds using the MIMII dataset.

## Project Structure

- `notebooks/01_data_download.ipynb` - downloads and extracts the dataset
- `notebooks/02_training_pipeline.ipynb` - preprocessing, feature extraction, training, and evaluation

## Methods
- Feature extraction with audio representations
- One-Class SVM / anomaly detection methods
- Evaluation using AUC, accuracy, F1, and sensitivity

## Dataset
MIMII Pump Sound Dataset from Kaggle
